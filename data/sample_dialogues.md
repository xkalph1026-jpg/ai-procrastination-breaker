# AI拖延症破解器 — 测试对话样本

## 测试目的

验证Skill在真实对话场景中的表现，覆盖6大拖延类型 + 模式识别的回归测试。

---

## 对话1：完美主义型拖延

```
👤 用户: 我不想写论文，已经拖了三天了
🤖 AI:   听起来卡住了。能说说具体是卡在哪一步吗？是不知道写什么内容，还是担心写得不够好？
👤 用户: 担心写得不够好，导师肯定不满意
🤖 AI:   我发现了一个规律：你在面对「创造性输出」任务时会触发完美主义。
        初稿的「初」字本来就可以很烂，海明威都说第一稿都是垃圾。
        来，试一个5分钟微行动：
        📌 打开Word → 只写标题和姓名学号 → 关掉。
        5分钟。就做这一个动作。
👤 用户: 做完了
🤖 AI:   ✅ 突破成功！
        📊 已记录你的战果：面对论文初稿，从0到有东西了。
        🧠 发现的模式：你容易在「需要创造新内容」的任务上卡住。
        下次再有类似感觉，记得告诉自己：「先造垃圾，再装修」。
        📅 明天同一时间继续？我可以帮你定下一个微行动。
```

## 对话2：任务模糊型拖延

```
👤 用户: 下周要交一个数据分析报告，但我连要分析什么都不知道
🤖 AI:   先别慌。原始数据有了是个好开头。
        你觉得现在最让你头大的是：
        A) 不知道从哪个角度分析
        B) 不知道用什么工具
        C) 不知道报告该怎么写
👤 用户: A，不知道从哪个角度分析
🤖 AI:   明白了，任务太模糊导致不知道第一步做什么。
        来个5分钟微行动：
        📌 打开原始数据 → 只看前20行 → 列出你看到的3个字段/维度 → 关掉。
        不需要分析，只观察。
👤 用户: 看了，字段有：日期、销售额、品类、地区、客户类型
🤖 AI:   ✅ 做得好！
        🧠 你刚才列出的5个字段就是5个分析角度。比如「哪个品类卖最好？」「不同地区有什么区别？」
        这就解决了"从哪分析"的问题。
        📊 已记录：你面对模糊任务需要先「具象化接触」才能启动。
```

## 对话3：任务过大型拖延

```
👤 用户: 期末要复习5门课，每门都好多内容，完全不知道从哪开始
🤖 AI:   5门课确实多，一口气想全吞掉谁都会崩溃。
        你心里最有底和最没底的分别是哪门？
👤 用户: 数学最有底，英语最没底
🤖 AI:   那就先解决最怕的。
        📌 只拿英语课本 → 翻到目录 → 圈出最陌生的3个单元 → 关上。
        10分钟够了。先解决最怕的，后面就轻松了。
👤 用户: 圈完了，第4、7、12单元
🤖 AI:   ✅ 完成！
        🧠 5门课 → 英语 → 3个单元。你刚才完成了最难的拆分。
        📊 已记录：你面对大量任务时容易焦虑性瘫痪，拆分后就能启动。
        明天微行动：只看第4单元的第一小节。
```

## 对话4：缺乏动力型拖延

```
👤 用户: 要填一个巨无聊的表格，两百多个字段，看到就烦
🤖 AI:   两百个字段确实反人类。不过——
        填完这张表的结果是什么？拿到实习机会之后你最期待的是什么？
👤 用户: 想去那个公司看看真正业界怎么做的
🤖 AI:   好，那这张表就是通往那个公司的门票。
        📌 找到表格里最容易填的10个字段（姓名、学号、电话...）
        → 填完它们 → 计时（我猜3分钟）→ 关掉。
        你离想去的公司又近了一步。
👤 用户: 填完了，真的3分钟
🤖 AI:   ✅ 3分钟就搞定了前10个！
        🧠 这不是懒，是你需要看到任务与目标之间的连接。
        📊 已记录：面对重复性任务时，给你一个「意义锚点」+ 极小量启动最有效。
        200个减10个 = 还剩190个。下次继续挑战10个？
```

## 对话5：决策疲劳型拖延

```
👤 用户: 老师让我们自选课题做项目，一周了还没定下来，看了十几个方向每个都想试试
🤖 AI:   收集了十几个方向说明你做事用心。现在卡在"选择"上了。
        三个问题帮你缩小：
        1️⃣ 哪个方向你能找到最多的现成资料？
        2️⃣ 哪个方向你愿意连续做两周不腻？
        3️⃣ 哪个方向做完能让你简历上多一行亮点？
👤 用户: A方向资料最多，B方向我感兴趣但不熟，C方向简历加分最多
🤖 AI:   问最后一个问题：不选哪个你会最后悔？
        这就是你的答案。
        📌 今天微行动：给这个方向写一句"一句话描述"发给老师确认。
        其他的不是你放弃，是「暂时存档」。
👤 用户: 选了B，一句话发老师了，老师说可以
🤖 AI:   ✅ 方向锁定！
        🧠 发现规律：你在决策时需要「缩小到2-3个」再用「最小后悔原则」就能定下来。
        📊 已记录：你容易因FOMO导致决策瘫痪。
        明天微行动：给B方向列出第一个小里程碑。
```

## 对话6：模式识别——老用户回归

```
👤 用户: 论文写到第三章又卡住了，不想写了
🤖 AI:   我记得你上个月写第一章也卡住了，当时是因为担心写得不够好。
        这次第三章是不是也是同样的感觉——怕写得不如前两章？
👤 用户: 是的，第一章和第二章写得还行，第三章怕掉水准
🤖 AI:   你的完美主义模式又来了！这是你第3次因为「怕写得不够好」而卡住。
        老办法：写一个"烂版本"的第三章开头——故意写烂。
        你上次第一章也是这么破的，还记得吧？
        📌 5分钟：只写第三章第一段，越烂越好。这是给自己的草稿。
👤 用户: 写完了，确实写得烂
🤖 AI:   ✅ 烂版本也是版本！你又一次打破了完美主义。
        📊 已更新记录：完美主义模式出现3次，破解成功率100%。
        💡 下次别等我来提醒，你自己可以立刻识别的信号是：
        「当你想写但又删掉」——那就是完美主义来了。
```
